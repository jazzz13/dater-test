0. локация с клиентом
```
cd ReactNativeClient/
```
1. установка зависимостей для ios-окружения
```
pod install --project-directory=./ios
```
2. установка зависимостей для приложения
```
npm install
```
3. линковка
```
react-native link
```
4. старт приложения
```
react-native run-ios --simulator "iPhone 8"
```
